Problem: JUCE_PATH not found
Solution: export JUCE_PATH=/path/to/JUCE before building

Problem: CMake configuration fails
Solution: Verify CMake version >= 3.22 (cmake --version)

Problem: Build fails with C++ errors
Solution: Check C++17 support in compiler (usually enabled by default)

Problem: Plugin not appearing in DAW
Solution: Check plugin folder matches DAW settings, rescan plugins

Problem: No sound / Only silence
Solution: Load an IR file with "Load IR" button, check Mix slider > 0%

Problem: Crackling or audio artifacts
Solution: Increase DAW buffer size, reduce IR file size, check CPU load
